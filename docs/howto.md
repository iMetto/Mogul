# Chat Usage Notes

This file is for your own chat/session habits. It is not an agent rule file.

## Usage Tracking

Check chat usage at the start of a work session, after broad research, and
before switching tasks. If usage is high, start a fresh chat with a short
handoff instead of relying on compaction.

Handoff format:

```text
Task: <one-line goal>
Done: <files touched and decisions made>
Next: <next concrete step>
Open: <questions or risks>
```
